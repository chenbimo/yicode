## 概述说明

`yitool-cli` 是一个用于增强开发体验和效率的工具库集合，旨在给开发者，带来方便。

## 使用声明

> 本项目使用 Apache 2.0 协议开源，可免费无偿使用及商用，需保留原作者版权信息。

## 功能特性

> 更多功能，敬请期待，欢迎反馈！

-   [x] git 项目提交统计
-   [x] 一键切换 npm 镜像
-   [x] 文件文件压缩为 zip 包
-   [ ] ftp 文件上传部署
-   [ ] 增加一键同步淘宝镜像缓存功能
-   [ ] git 统计显示项目名称相关信息
